## 7. Anki card recipes

**Application**

```
Q: [3 concrete facts]. What issue arises and what test applies?
A: Issue → Rule/test → Application sketch → Authority. Distinguishing fact.
```

**Distinction**

```
Q: What factual change would allow a court to distinguish Authority A from B?
A: …
```

**Obiter / ratio**

```
Q: Why is proposition P arguably obiter in [Case]?
A: …
```

**Deck mix:** 15% definitions · 25% case–proposition · 25% distinctions · 35% application.

**FSRS:** desired retention 0.85–0.90; re-optimise monthly.

## 13. Offer diligence script

### Law firm
- PCLL tuition funded? Maintenance grant? Repayment triggers?
- Recent trainee → NQ retention?
- How are seats allocated?
- Personal account dealing / restricted list policy?
- Bonus determination?
- Realistic secondments?

### Quant
- Guaranteed vs discretionary? Deferral / clawback?
- What counts as attributable P&L?
- Non-compete / IP ownership?
- Personal trading rules?
- What happens after a drawdown?

### VC
- Exact carry % / points? Fund- vs deal-based?
- Vesting? Bad leaver? Clawback? Co-invest?
- Deployment pace? Dry powder? DPI / realised exits?
- Investing vs portfolio support vs fundraising?
- Promotion path?
